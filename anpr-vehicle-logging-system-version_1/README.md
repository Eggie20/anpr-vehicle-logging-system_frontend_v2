# CampusSecure ANPR System

CampusSecure is an advanced Automated Number Plate Recognition (ANPR) vehicle logging system designed for campus security. It provides a comprehensive solution for managing vehicle access, tracking entry/exit logs, and ensuring campus safety through real-time monitoring.
